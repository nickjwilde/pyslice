Learn

This is the entry page for learning different skills and techniques you will need as a data scientist. I've laid out the road map below to give you an overview of what topics are covered and in what order you can proceed. You'll be following the same order I have followed in my journey learning data analytics.

Part 1

Satistics

Here you will learn the difference between descriptive and inferential statistics. This is the foundation you'll build other topics and ideas on.

  • Brick, Mortar, and Wood: Laying the Foundation

Part 2

Maths and Stuff

In this section you will learn the math needed to excel as a data analyst. We will cover topics in Linear Algebra and Multivariable Calculus

  • In The Beginning, There Was a Dot
  • Afraid of the Unknown

Part 3

Databases

In Part 3, you will learn SQL syntax and how to handle databases. You will learn about NoSql and how it's used in data warehousing. We'll also touch on tools like Hadoop and Spark.

  • A Sequel to Data
  • Rise of the NoSql
  • The Right Tool for the Job

Part 4

Visualization

During Part 4, You will learn about the different techniques and tools to visualize the data in a meaningful way.

Part 5

Machine Learning
