Why Pyslice?

Pyslice came about because I wanted a place that made it easy to learn Python. I love using Python, not only as a hobby language, but also at work. Python has terse syntax which makes it easy to type, and when you are a programmer, typing is something you do most of your day.

Python has an interactive interpreter, which comes in handy when you want to prototype something or try something out quickly. This allows you to throw out ideas without having to invest a lot of time in them. Python is easy to start using. You don't really have to install a large amount of software. Although the selection of IDEs is few, you don't really need one to start with Python.
